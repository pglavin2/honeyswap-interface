# Honeyswap

An open source AMM forked from Uniswap v2 by 1Hive, deployed on Gnosis Chain. Swap and provide liquidity for tokens on Gnosis Chain.

## Development

### Install Dependencies

```bash
yarn
```

### Run

```bash
yarn start
```
